
# TaskManager API

A Task Management API built using **ASP.NET Core** following **Clean Architecture** principles.

## Features

* Create, update, delete tasks
* Mark tasks as completed
* RESTful API design
* Clean Architecture structure
* Unit testing

## Tech Stack

* ASP.NET Core
* Entity Framework Core
* PostgreSQL
* Redis caching

## Architecture

* Domain
* Application
* Infrastructure
* Web (API)

## Run Locally

```bash
dotnet restore
dotnet build
dotnet run --project TaskManager.Web
```
